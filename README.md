# tomis-lang

Experiment, simple,

not supposed to be a full language.

Transpiler to lua.

Uses a stack to pass parameters to functions.

## Examples

Check the examples folder.

## Transpiler to lua

Transform tomis code to lua.

Example:

    lua tomis2lua.lua input.t output.lua
