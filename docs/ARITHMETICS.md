# stack arithmetics

Arithmetics:

Expects 2 elements on the current stack, pops the 2 elements and push the result.

    + - addition        : ( n1 n2 — sum )
    - - substraction    : ( n1 n2 — diff )
    * - multiplication  : ( n1 n2 — prod )
    / - division        : ( n1 n2 — quot )
    % - mod             : ( n1 n2 — rem )

Alternative, operation with param will push the number and execute the op.
